# Warp OS for Raspberry Pi 3 #

### What is this repository for? 
Control arduino and iRobot create2 from remotely using Raspberry Pi.

![yellowLine](client/assets/images/yellowLine.png)

### How do I get set up? ###

```sh
$ git clone path
$ cd path
$ npm install
$ npm start
```
